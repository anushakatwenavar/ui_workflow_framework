# UI Workflow Recorder Framework

An enhanced Playwright-based workflow recorder that:
- Logs in automatically
- Records all browser interactions
- Captures API requests
- Generates ready-to-run Locust load test scripts

## Installation
```bash
pip install -r requirements.txt
playwright install chromium



python -m cli.main #to run the recorder first run this and then record 

#after you close the browser it will give you the command to run locust
